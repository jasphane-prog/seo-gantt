⚙️ Google Sheets 設定步驟

  1. 前往 sheets.new 建立新試算表
  2. 第一列填入欄位名稱（已預設）：id task plannedStart plannedEnd actualEnd status note
  3. 從第二列開始填入任務資料
  4. 點選右上角「共用」→ 將連結設為「知道連結的人可以檢視」
  5. 複製試算表 URL 中的 spreadsheetId（/d/ 和 /edit 中間的部分）
  6. 將 ID 貼到下方設定欄位，按 Enter 或點「載入」
  7. 新增/編輯需填入 Google Sheets API Key（取得方式）
    或直接在 Google Sheets 中手動編輯，本頁面點「重新整理」即可同步

💡 若只需查閱，僅需 Sheet ID 即可（唯讀模式）。寫入功能需 API Key。
